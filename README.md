Chord Practice
==============

A simple tool for guitar beginners to practice chord transitions. It
prints random chord transitions at tunable intervals, showing chord
patterns as they happen.

Example Output (one chord)
==========================

    Chord Am (x02210)
    
      E A D G B E
        o       o
      -----------
    1 | | | | * |
    2 | | * * | |
    3 | | | | | |
    4 | | | | | |
    5 | | | | | |
